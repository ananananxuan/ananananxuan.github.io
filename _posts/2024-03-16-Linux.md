# 相对路径和绝对路径
1. 相对路径 从根目录开始 cd /
2. 绝对路径 以当前目录为起点
3. 当前目录 .
4. 上一级目录 .. cd ..
5. ~， cd ~ 表示进入\home\hax

## 2.Home目录 在根目录下面/home

# Linux命令基础
命令本体 -选项 参数
## 1.ls [-l/a/h] [路径]
ls 列出当前目录下的内容，默认是/home
-a 显示所有文件，包括隐藏文件  
-l 详细信息，以列表形式展示  
-al 显示所有文件，包括隐藏文件，且成列表形式展示
-lh 文件大小以K/M/G的形式展示

## 2.cd [路径]
cd 切换目录 change directory
默认情况下cd 是 cd /home
## 3.pwd
pwd 打印当前工作目录的完整路径 print working directory

## 4.mkdir [-p] 路径
-p 递归创建目录
需要在home目录内
